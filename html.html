    python vs C언어

    파이썬이 더 간단한 느낌.
    코드를 구별할 때 , () 등을 사용

    C언어는 조금 더 복잡한 느낌.
    ; {} 등을 사용

    공통점: 변수, 논리구조 등

    html에 대한 간단한 조사

    동기: HTML을 배우며 c언어, 파이썬을 배우던 구조와는 다르다는 생각이 들었다. 변수나 함수 같은 건 언제 배우지? 라는 궁금증이 들어서 직접 찾아보았다.
    약자: Hypertext Markup language

    마크업 언어란?

    - 태그 등을 이용해 문서나 데이터의 구조를 명기하는 언어의 한 가지

    HTML은 기술적 마크업 언어이다. 마크업 언어는 페이지의 틀을 만드는 언어로, 변수/ 논리구조 등의 요소가 없고, 정적이다. Javascript등의 프로그래밍 언어가 웹페이지를 동작하게 하는 것이다.
